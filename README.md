# 課題（todo-app）

## 要件一覧
- TODOを追加できること
    - 項目1：タイトル
    - 項目2：期日
- TODOを一覧表示できること
- 各TODOに完了・未完了のチェックができること
- 各TODOのタイトル、期日を変更できること
- 各TODOを削除できること
- TODOを状態ごとにフィルタリングできること（一覧・完了・未完了）
- TODO一覧をタイトルでフィルタリングできること(前方一致・中間一致・後方一致・完全一致のどれでもOKです。)

※データの保存機能は不要です。画面リロードしたら登録していたTODOは消えていてOKです。
